Aufgabe: Listen

Nun sollen Sie die erste größere Aufgabe dieser Lektion bearbeiten. Dafür implementieren Sie bitte das Sortierverfahren Bubblesort, welches eine gegebene Liste in die richtige Reihenfolge bringen soll (Zahlenwerte aufsteigend sortiert).

Bei Bubblesort wird jeweils in Runden für jedes benachbarte Paar von Elementen getestet, ob diese vertauscht werden müssen. Sobald mindestens eine Vertauschung notwendig war, beginnt eine neue Runde. War keine Vertauschung notwendig, dann ist die Liste korrekt sortiert.


Von Swfung8-Eigenes Werk,CC BY-SA 3.0,Link

Seinen Namen hat das Sortierverfahren der Bewegung der größeren Elemente nach und nach von links nach rechts zu verdanken. Sie steigen somit ähnlich wie Blasen im Waser nach oben.

Ordnen Sie nun also die gegebene Liste unordered_list mithilfe des Sortierverfahrens.

Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run ausführen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen (Löscht auch Ihren Code!). Außerdem wird eine Musterlösung für die Aufgabe durch einen Klick auf den Button Solution bereitgestellt.

Pseudocode

unordered := True
WHILE unordered:
 unordered := False
 FOR counter := TO LENGTH(list) - 1 DO
  IF counter < LENGTH(list) - 1
  THEN IF list[counter] > list[counter + 1]
     THEN temp := list[counter]
        list[counter] := list[counter + 1]
        list[counter + 1] := temp
        unordered := True
Bitte Code hier eingeben:

# In den folgenden Zeile bitte ihren Code eintragen

unordered = True

while unordered:
 unordered = False

 for counter in range(len(unordered_list) - 1):
  if counter < len(unordered_list) - 1:
    if unordered_list[counter] > unordered_list[counter + 1]:
     temp = unordered_list[counter]
     unordered_list[counter] = unordered_list[counter + 1]
     unordered_list[counter + 1] = temp
     unordered = True

# Hiermit wird die Liste ausgegeben

print(*unordered_list)
