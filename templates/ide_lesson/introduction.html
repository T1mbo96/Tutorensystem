Einführung

Zum Abschluss dieses Tutorials dürfen Sie nun Aufgaben in einer eigens gewählten IDE lösen. Hierfür wird Ihnen ein Guide zur Installation der IDE PyCharm bereitgestellt. Sie können entweder zwischen der Community Edition, welche nicht alle Features enthält, oder der Professional Edition wählen. Die Professional Edition ist eigentlich kostenpflichtig, jedoch können Sie sich mithilfe Ihrer Mail-Adresse der Universität für einen kostenfreien Zugang registrieren. Auch dies wird im nachfolgenden Guide beschrieben.

Die jeweiligen Schritte zur Installation der Komponenten wurden auf einem Windows-System durchgeführt. Details dieser Erklärungen könnten auf anderen Betriebssystemen abweichen.
