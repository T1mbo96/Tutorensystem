Bedingte Anweisungen: Zusatz

Sie haben bedingte Anweisungen und ihre Funktionsweise bereits kennenlernen dürfen. Nun wird dieses Konstrukt um einen else if-Teil (elif in Python) erweitert. Mit diesem kleinen Zusatz können Sie mehrere unterschiedliche Bedingungen innerhalb eines einzelnen if-Blocks abarbeiten, ohne jedesmal einen neuen if-Block benutzen zu müssen.

Wie sie hier erkennen können, dürfen Sie die bedingte Anweisung mit einem Klick auf den blauen Button beliebig anpassen. Außerdem können Sie nun die im vorherigen Abschnitt erlernten Variablen benutzen, um die Bedingung selbst zu erstellen und nicht mehr die vorgefertigten Anweisungen (z.B. is path to the left) verwenden zu müssen, z.B.

Wenn Sie diesen Block nun als Bedingung in einer if-Anweisung verwenden würden, dann würde dies einfach wenn x kleiner als 50 ist bedeuten.

Der erste Teil if -> do würde ausgeführt werden, wenn x kleiner als 50 ist. Der Zweite Teil else if -> do würde ausgeführt werden, wenn x größer oder gleich 50 ist und y größer 80 ist. Der dritte Teil else würde nur dann ausgeführt werden, wenn x größer oder gleich 50 ist und y kleiner oder gleich 80 ist.
