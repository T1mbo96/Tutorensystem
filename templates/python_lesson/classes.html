Klassen

Klassen stellen in der objektorientieren Programmierung die Baupläne für Objekte dar. Sie können mithilfe dieses Konstrukts eigens definierte Datenstrukturen erstellen und sind somit nicht mehr nur an die von Python standardmäßig bereitgestellten Datenstrukturen (z.B. Listen oder int) gebunden.

Zur Veranschaulichung können Sie sich vorstellen, dass Sie z.B. eine größere Menge an Tieren in Ihrem Programm abbilden möchte, die jeweils mehrere Attribute wie Bezeichnung, Größe, Gewicht, etc. besitzen. Sie könnten dafür jetzt möglicherweiße ein Dictionary heranziehen, allerdings wird dies sehr schnell sehr unübersichtlich und sehr komplex. Hierfür bieten sich nun Klassen an, sodass Sie einfach eine Klasse Animal erstellen, welche mehrere gewünschte Attribute beinhaltet. Mithilfe dieses erstellten Bauplans können Sie nun für jedes der Tiere eine Instanz erstellen, also ein Objekt. Mit diesem können Sie anschließend weiterarbeiten.

Eine Klasse ist nur der Bauplan, während eine Instanz eine Kopie der Klasse ist, welche tatsächliche Werte wie z.B. 'Hund' oder '1.20' für die Größe enthält.

Zur Veranschaulichung wird im Folgenden sukzessive die Klasse Animal aufgebaut werden.

Definition

Eine Klasse können Sie mithilfe des Schlüsselworts class und dem gewünschten Namen definieren.

Das Schlüsselwort pass dient in Python als Platzhalter. Dieser Platzhalter macht es möglich den Code ohne Logik auch ohne Fehler laufen zu lassen.

Instanzattribute

Um nun die Logik für die Attribute, welche für jedes Tier unterschiedlich sind, zu implementieren, verwenden wir sogenannte Instanzattribute.

Diese werden in einer Funktion namens __init__ definiert. Diese Funktion wird aufgerufen, wenn eine neue Instanz der Klasse erstellt wird und weißt die übergebenen Werte (oder die Standartwerte, falls kein Wert übergeben wurde) dem jeweiligen Instanzattribut zu.

Klassenattribute

Im Gegensatz zu Instanzattributen, sind Klassenattribute für jede Instanz der Klasse exakt gleich.

Diese werden einfach direkt nach der Definition der Klasse implementiert.

Instanziierung

Damit Sie nun viele verschiedene Objekte einer Klasse verwenden können, müssen Sie jedes Objekt zuerst instanziieren.

Sie instanziieren ein Objekt einer Klasse, indem Sie einem selbst gewählten Namen Klassennamme(Wert(e)) zuweisen. Hier zu erkennen ist, dass Sie Attribute, welche Standardparameter haben, nicht übergeben müssen.

Instanzfunktionen

Sie können in einer Klasse nicht nur Variablen bzw. Attribute, sondern auch ganze Funktionen implementieren.

Diese werden genau so definiert, wie Sie es im Abschnitt Funktionen kennengelernt haben. Wichtig ist, dass Sie bei der Definition der Parametern immer zuerst das Schlüsselwort self implementieren. Dieses müssen Sie beim Aufruf der Funktion aber nicht übergeben, wie Sie aus dem vollständigen Beispiel entnehmen können.

Modifizierung der Attribute

Sie können die Attribute eines Objekts im Nachhinein modifizieren, dafür implementieren Sie eine Funktion innerhalb der Klasse, um dem gewünschten Attribut einen neuen Wert zuzuweisen.

Diese Funktionen werden analog zu Instanzfunktionen implementiert.
