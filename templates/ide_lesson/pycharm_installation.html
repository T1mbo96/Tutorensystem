PyCharm Installation

PyCharm ist eine lokale IDE, welche von JetBrains entwickelt wird. Angeboten wird eine kostenfreie Community-Edition, welche weniger Features enthält, oder eine kostenpflichtige Professional-Edition, welche den vollen Umfang an Features enthält. Durch die Einschreibung an einer Universität wird Ihnen die Professional-Edition für den Zeitraum Ihres Studiums konstenlos zur Verfügung gestellt, weshalb empfohlen wird diese herunterzuladen. Wichtig ist, dass Sie wieder die passende Version für Ihr Betriebssystem auswählen.

Studenten-Lizenz

Um sich für die kostenlosen Produkte von Jetbrains zu registrieren, müssen Sie dem folgenden Link zur Aktivierungsseite von Jetbrains folgen und dort die entsprechenden Angaben machen.

Link zur Aktivierung

Sie müssen nur die Felder entsprechend ausfüllen und im rot marktierten Feld ihre E-Mail-Adresse der Universität angeben (z.B. vorname.nachname@stud-mail.uni-wuerzburg.de). Anschließend wird Ihnen eine Bestätigungsmail an Ihre Adresse gesendet, in denen Anweisungen zum weiteren Vorgehen aufgeführt sind.

Installation

Download-Link

Wählen Sie die entsprechende Version aus und laden Sie diese herunter. Anschließend starten Sie die Installation mithilfe der entsprechenden Datei und folgen Sie den Anweisungen.

Setzen Sie bei allen rot markierten Feldern einen Hacken und folgen Sie danach weiter den Anweisungen des Setups. Nach erfolgreicher Installation starten Sie das Programm. Nun müssen Sie ihre IDE nurnoch aktivieren.

Geben Sie hier einfach Ihre Universitätsmail und Ihr Passwort an und klicken anschließend auf Activate.
