Dictionary

Ein Dictionary ist ein etwas anderer sequentieller Datentyp. Dieses speichert Werte als sogenannte Schlüssel-Wert-Paare. Das heißt, dass man auf einen Wert eines Dictionarys nicht mithilfe eines Index, sondern mithilfe eines Keys zugreift.

Wie zu erkennen ist, werden Dictionarys ebenfalls mit geschweiften Klammern { } erzeugt. Allerdings wird immer jeweils ein Key und ein Schlüssel, getrennt durch einen Doppelpunkt, angegeben.
