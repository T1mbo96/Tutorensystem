Variablen

In der Programmiersprache Python bezeichnet eine Variable, analog zu den Spielen in Blockly, keinen bestimmten Typ, weshalb ebenfalls keine Deklaration benötigt wird. Benötigt man im Programm bespielsweise eine Variable i mit dem Wert 50, so erreicht man dies einfach mit der folgenden Anweisung.

Aus dem oben genannten Grund können Variablen in Python ihren Typ sogar ändern, nachdem sie festgelegt wurden. Ein Beispiel dafür ist in folgendem Bild ersichtlich.
