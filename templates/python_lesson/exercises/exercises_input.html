Aufgabe: Input-Funktion

Lesen Sie bitte mithilfe der Input-Funktion den Wert 532 von der Konsole ein und speichern Sie diesen in der Variable x. Sie müssen den Wert in der aufgehenden Alert-Box eintragen, wenn Sie ihren Code laufen lassen. Diese Alert-Box soll hier die Konsole darstellen. Der Test wird nur bei Eingabe des korrekten Werts von 532 erfolgreich sein, allerdings können Sie gerne auch andere Werte ausprobieren.

Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run testen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen.
