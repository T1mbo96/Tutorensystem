Input-Funktion

Wie Sie im vorherigen Abschnitt kennengelernt haben, muss jedes Programm mit seiner Außenwelt in Kontakt treten. Sie haben bereits in Erfahrung bringen können, wie man mithilfe der Print-Funktion Werte in der Konsole ausgibt. Nun lernen Sie, wie man von der Konsole Werte einlesen und mit diesen arbeiten kann.

Da Skulpt momentan eigentlich noch für Python 2 konzipiert wurde und noch nicht die komplette Funktionalität von Python 3 aufweist, muss dieser Abschnitt unterteilt werden. Machen Sie sich allerdings keine Sorgen, die Funktionen in Python 2 und Python 3 weisen im Endeffekt dieselbe Funktionsweise auf, nur die Namen haben sich geändert. Die Print-Funktion im vorherigen Beispiel ist übrigens schon aus Python 3, da hierfür bereits das Modul für Skulpt programmiert wurde. Leider ist dies für die Input-Funktion noch nicht geschehen.

Python 2

In Python 2 werden die Werte mithilfe der Funktion input() von der Konsole eingelesen, welchen direkt ihr passender Datentyp zugewiesen wird. Möchte man jeglichen Input als String haben, so benutzt man die Funktion raw_input().

Python 3

In Python 3 benutzt man für das Einlesen als String die Funktion input() (= raw_input() in Python 2). Um den eingelesenen Werten nun direkt die richtigen Datentypen zuzuweisen, muss man die Funktion eval() benutzen. Diese Funktion benötigt aber im Gegensatz zur Input-Funktion in Python 2 noch die Funktion input() als Parameter, woraus sich eval(input()) ergibt.

Wenn Sie die Input-Funktion in Skulpt verwenden wollen, dann benutzen sie bitte die Python 2 Version.
