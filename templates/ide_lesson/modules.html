Module

Module sind eine weitere Art Ihren Code logisch zu organisieren. Durch das Gruppieren von verwandtem Code in einem Modul wird der Code verständlicher und benutzerfreundlicher. Außerdem gibt es für Python eine riesige Auswahl an Modulen, die von anderen Entwicklern programmiert wurden und welche Sie für Ihren Code benutzen können. Somit müssen Sie das Rad nicht jedes Mal neu erfinden und können einfach bereits geschriebene Logik wiederverwenden.

Ein Modul ist also eine einfache Datei, welche Python-Code enthält. Es kann Funktionen, Klassen und Variablen definieren und ebenfalls lauffähigen Code beinhalten.

PyCharm

Um ein Modul in einer lokal IDE verwenden zu können, müssen Sie zuerst das entsprechende Modul auswählen, welches Sie nutzen wollen und dieses anschließend installieren. In diesem Beispiel wird das gesamte Vorgehen am Modul fibonacci vorgeführt. Dieses Modul enthält Code zur Berechnung der n-ten Fibonacci Zahl.

Damit Sie das Modul nun in ihrem Code nutzen können, müssen Sie es zuerst importieren. Hierfür verwendet man das Schlüsselwort import, gefolgt vom Namen des jeweiligen Moduls.

Wie hier zu erkennen ist, wird das Modul von der IDE noch nicht erkannt, da das Modul noch nicht installiert wurde. Im nächsten Schritt setzen Sie Ihren Cursor auf den rot markierten Modulnamen und klicken auf die erscheinende rote Glühbirne (wahlweise können Sie auch Alt + Enter drücken).

Nun wählen Sie Install package fibonacci, wodurch das entsprechende Modul automatisch installiert wird. Anschließend sollte Ihnen das Modul als installiert angezeigt werden und nicht mehr rot markiert sein. Nun können Sie den Code aus der Datei verwenden.

In diesem Beispiel haben wir jetzt aus dem Modul fibonacci die Funktion fibo() verwendet und die 20. Fibonacci Zahl berechnen lassen.

Der Code im Modul fibonacci sieht folgendermaßen aus. Dieses Modul stellt ein einfaches Python-File mit Code einer einzigen Funktion dar.

Somit haben Sie die Logik zum Berechnen der n-ten Fibonacci Zahl nicht selbst implementieren müssen, sondern konnten sich eine Menge Zeit sparen und einfach bereits vorhandenes wiederverwenden.
