Python Installation

Zuallererst ist es wichtig, dass Sie Python auf ihrem Computer installieren. Dafür laden Sie sich bitte die jeweilige Datei, passend zu Ihrem Betriebssystem, von folgendem Link herunter.

Download-Link Python

Nach erfolgreichem Download, starten Sie bitte die Installation mit einem Doppelklick auf die entsprechende Datei und folgen Sie den Anweisungen.

Setzen Sie hier einen Haken bei Add Python 3.7 to PATH und wählen Sie anschließend Install Now aus (Die Versionsnummer 3.7 kann natürlich mit neueren Releases abweichen).

Um zu testen, ob die Installation erfolgreich war, öffnen Sie ein Terminal und geben 'python' ein. Anschließend sollten sie etwas Ähnliches zu Folgendem sehen.
