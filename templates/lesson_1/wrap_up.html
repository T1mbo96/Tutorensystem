Zusammenfassung

Herzlichen Glückwunsch!

Sie durften sich in der ersten Lektion von Learn_Python durch mehrere Spiele kämpfen, die sie mithilfe von Blockly bearbeiten konnten. Dabei haben Sie grundlegende Programmierkonstrukte, wie zum Beispiel Schleifen, bedingte Anweisungen oder Funktionen kennengelernt, die Ihnen in den nächsten Lektionen helfen werden eigenen Code in Python zu schreiben.

In der zweiten Lektion von Learn_Python werden Sie nun alle relevanten Konstrukte in Python wiederholen bzw. neue erlernen. Auch werden Sie sich mit der Syntax, also den grundlegenden Regeln der Schreibweise einer Programmiersprache, beschäftigen, um nach Abschluss dieser zweiten Lektion selbstständig die Aufgaben in Lektion 3 in einer richtigen IDE (Integrated Development Environment) zu bearbeiten.

Sie können entweder unten auf den Button Next klicken oder oben im Menü Lektion 2 auswählen, um zur nächsten Lektion zu gelangen.
