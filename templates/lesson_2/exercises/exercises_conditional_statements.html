Aufgabe: Bedingte Anweisungen

Schreiben Sie für die nachfolgendee Tabelle eine bedingte Anweisung, sodass ein Wettbewerber wissen kann, welchen der Preise aus der Tabelle er gewonnen hat, basierend auf der Anzahl der erzielten Punkte, die in der ganzzahligen Variable points gespeichert ist.

Alle unteren und oberen Grenzen sind hier inklusive und Punkte können nur positive ganzzahlige Werte bis zu 200 annehmen.
Weisen Sie in Ihrer bedingten Anweisung der Variable result eine Zeichenfolge (String) zu, die, basierend auf dem Wert der Punkte, die entsprechende Nachricht enthält. Wenn ein Preis gewonnen wurde, sollte die Nachricht Sehr gut! Sie haben einen [Preisname] gewonnen! (Ersetzen Sie [Preisname] mit dem Namen des jeweiligen Preises aus der Tabelle) ausgegeben werden. Wenn es keinen Preis gibt, sollte in der Nachricht Oh schade, diesmal gibt es keinen Preis. stehen.

Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run testen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test.

Sie dürfen ihren eigenen Code gerne mit anderen Eingaben testen. Um den Test bestehen zu können, sollten Sie zum testen nur den ursprünglichen Wert von 174 nutzen.
Bitte Code hier eingeben:

Klasse! Sie haben den richtigen Preis korrekt ausgegeben!
Schade, der Preis oder die Nachricht ist leider nicht korrekt!
