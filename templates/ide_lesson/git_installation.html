Git Installation

Git ist eine Software zur verteilten Versionsverwaltung von Dateien. Sie müssen diese installieren, damit Sie die Projekte von GitHub klonen können. Die Software ist kostenfrei und für Sie nach der Installation in diesem Tutorial nicht mehr direkt von Belang. Allerdings schadet es nicht, sich ein wenig mit der Software auseinanderzusetzen, da Sie eine der am häufigsten verwendeten Versionsverwaltungen ist.

Download-Link Git

Wählen Sie aus den Optionen erneut die passende für Ihr Betriebssystem.
Nachdem Sie die Installation gestartet haben, folgen Sie einfach den Anweisungen des Setups. Sie können bei allen Auswahlmöglichkeiten die standardmäßig angewählten Optionen belassen. Wenn Sie die Installation abgeschlossen haben, können Sie sie auf Erfolg überprüfen, indem Sie erneut ein Terminal öffnen und git eingeben. Sie sollten nun etwas Ähnliches zu Folgendem sehen.
