Aufgabe: Operatoren

Hier werden verschiedene Aufgaben zu Ausdrücken mit Operatoren zur Verfügung gestellt. Sie bekommen jeweils einen Ausdruck gestellt und sollen das richtige Ergebnis aus insgesamt vier gegebenen Möglichkeiten auswählen.

Mit einem Klick auf den Button Check können Sie Ihre Antwort testen. Bei einer richtigen Antwort färbt sich der jeweilige Abschnitt grün, bei einer falschen Antwort färbt sich der jeweilige Abschnitt rot.
5 + 3 * 7 // 3



5 + 3 > 7 // 3



21 == 34 is 5 // 2



7 % 112 <= 89



87 % 25 // + 56 % 23



27 in [27, 23, 25]
