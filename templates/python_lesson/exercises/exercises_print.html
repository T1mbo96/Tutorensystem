Aufgabe: Print-Funktion

Geben Sie bitte zuerst die Variablen one und two mithilfe der Print-Funktion standardmäßig in einem Funktionsaufruf aus. Anschließend sollen die Variablen one und three mit einem & (und) getrennt ausgegeben werden. Zum Schluss soll dann noch die Variable one mit einem ! am Ende ausgegeben werden, ohne Zeilenumbruch.

Die Reihenfolge der Ausgabe ist wichtig! Also arbeiten Sie die Aufgabe bitte von oben nach unten durch und geben Sie bitte das Geforderte auch in dieser Reihenfolge aus!
Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run ausführen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen (Löscht auch Ihren Code!). Außerdem wird eine Musterlösung für die Aufgabe durch einen Klick auf den Button Solution bereitgestellt.
Bitte Code hier eingeben:

# Hier werden die Variablen erzeugt

one = "Variable"
two = 10
three = 3.5

# Geben Sie nun die Variablen den Anweisungen folgend aus

print(one, two)
print(one, three, sep="&")
print(one, end="!")
