Funktionen

Auch Funktionen haben Sie bereits in der ersten Lektion kennengelernt. Zusammengesetzt ist eine Funktion aus einem optionalen Rückgabewert, dem Funktionsnamen und den optionalen Parametern, welche die Funktion erhält, um damit zu arbeiten.

Eine Funktion wird in der Programmiersprache Python mit dem Schlüsselwort def erzeugt.

if __name__ == '__main__': stellt eine besondere Funktion dar, nämlich die Main-Funktion. Diese Funktion dient als Eintrittspunkt in ein Programm. Mehr Informationen zu dieser können Sie folgendem Link entnehmen.

In diesem Fall wurde die Funktion mit dem Funktionsnamen fizz_buzz mithilfe des Schlüsselworts def erzeugt. Die Funktion erhält einen Parameter number, welcher hier eine Zahl darstellt. Zurückgegeben werden Strings mithilfe des Schlüsselworts return.

Festlegen des Typs des Rückgabewerts

Sie können festlegen, welchen Typ Ihre Funktion zurückgeben soll (z.B. str oder int). Falls innerhalb der Funktion anschließend versucht wird, einen anderen Typ zurückzugeben, wird eine Warnung angezeigt.

Funktionen ohne Rückgabewert

Funktionen müssen, wie oben bereits erwähnt, nicht unbedingt einen Rückgabewert haben.

Beim Aufruf dieser Funktion würde kein Wert zurückgegeben, sondern einfach direkt der entsprechende String auf der Konsole ausgegeben werden.

Standardparameter

Sie können Parametern einer Funktion einen Standardwert zuweisen, sodass dieser Wert benutzt wird, falls beim Aufruf dieser Funktion für diesen Parameter kein Wert übergeben wird. Dafür müssen Sie hinter dem jeweiligen Parameter diesem einen Wert mit einem = zuweisen.

Pass by Reference

Alle Parameter in der Programmiersprache Python werden als Referenz übergeben. Das bedeutet, dass wenn Sie eine Variable an eine Funktion übergeben nicht der Wert der Variable, sondern die Referenz auf diese Variable übergeben wird. Am einfachsten kann man sich dieses Verhalten anhand eines Beispiels vor Augen führen.

Der Funktionsaufruf ändert somit die Variable my_list, sodass man bei einem Aufruf der Variable danach Zugriff auf die aktualisierte Variante der Variable hat.

Andere Programmiersprachen (z.B. C) übergeben ihre Parameter als Kopie. Diese Vorgehensweise trägt den Namen Pass by Value.

Wie Sie hier erkennen können, wird die Variable my_list zwar scheinbar innerhalb der Funktion change_value geupdated, allerdings wird in Wirklichkeit nur die Kopie der Variable aktualisiert, sodass die eigentliche Variable unverändert bleibt.
