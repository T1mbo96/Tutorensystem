Listen

Eine Liste ist ein sogenannter sequentieller Datentyp. Das bedeutet, dass in Listen ebenfalls mehrere Elemente gespeichert werden können, die über einen Index auswählbar sind (vgl. Strings). Der große Unterschied zu Strings ist jedoch, dass die gespeicherten Elemente nicht vom selben Typ sein müssen. Eine Liste wird mit eckigen Klammern [] initialisiert. Außerdem kann man, analog zu der Vorgehensweise bei Strings, auf die einzelnen Elemente mithilfe der Indizes zugreifen.

Im Folgenden werden einige Operationen und Methoden von Listen in Python vorgestellt. Für einen ausführlicheren Überblick benutzen Sie bitte die Referenz Python Listen 1.

Grundlegende Operationen

Bezeichnung Ausdruck Ergebnis
Länge len([1, 2, 3, 4, 5]) 5
Wiederholung ["Ha"] * 2 ["Ha", "Ha"]
Zugehörigkeit 3 in [1, 2, 3, 4, 5] True
Verkettung [5, 10] + [15, 20] [5, 10, 15, 20]

Iteration

Um alle Elemente eines sequentiellen Datentyps durchzugehen benutzt man die im vorherigen Abschnitt erlernte for-Schleife.

Die Bezeichnung element könnte man hierbei beliebig variieren.

Schneiden

Es kann auch auf mehrere Elemente einer Liste auf einmal zugegriffen werden. Dafür verwendet man einen Doppelpunkt : und die Schreibweise, um auf einen bestimmten Index zuzugreifen.

Hierbei ist zu erkennen, dass die Angabe vor dem Doppelpunkt immer inklusive und die Angabe nach dem Doppelpunkt immer exklusive ist. Somit kann man ebenfalls mehrere Elemente einer Liste auf einmal verändern.

Für Fortgeschrittene: Das Sternchen * vor dem Namen der Variable mit dem sequentiellen Datentyp (in diesem Beispiel eine Liste) liefert alle Elemente der Liste als iterable.

Append

Um einer Liste einen neuen Eintrag hinzuzufügen benutzt man die append-Methode. Mithilfe dieser wird das gewünschte Element ans Ende der Liste angehängt.

Delete

Man kann Elemente aus einer Liste ebenfalls wieder löschen. Dies geschieht mithilfe des Schlüsselworts del und der Auswahl des jeweiligen Index.
