Aufgabe: Strings

Erzeugen Sie bitte eine Variable mit Namen test_string, deren Wert Sie auf Hallo, Welt! setzen. Anschließend erzeugen Sie bitte drei weitere Variablen mit den Namen first_string, second_string und third_string. Weißen Sie first_string bitte den Wert des sechsten Elements, second_string den Wert des zweiten Elements und third_string den Wert des letzten Elements zu. Nutzen Sie zu Lernzwecken bitte die Zuweisung durch Indexierung der test_string Variable, wie Sie in vorhergehendem Abschnitt gelernt haben.

Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run testen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen.
Bitte Code hier eingeben:

Klasse! Die Variable 'test_string' ist korrekt!
Schade, die Variable 'test_string' ist leider nicht korrekt!
Klasse! Die Variable 'first_string' ist korrekt!
Schade, die Variable 'first_string' ist leider nicht korrekt!
Klasse! Die Variable 'second_string' ist korrekt!
Schade, die Variable 'second_string' ist leider nicht korrekt!
Klasse! die Variable 'third_string' ist korrekt!
