Aufgabe: Variablen

Im folgenden Textfeld können Sie sich nun selbst daran versuchen eine Variable zu erstellen, um mit der Syntax und Funktionsweise in Python vertraut zu werden. Erzeugen Sie bitte eine Variable mit Namen integer, deren Wert Sie auf 50 setzen. Anschließend wird die Variable mithilfe der print()-Funktion ausgeben.

Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run ausführen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen (Löscht auch Ihren Code!). Außerdem wird eine Musterlösung für die Aufgabe durch einen Klick auf den Button Solution bereitgestellt.

Bitte Code hier eingeben:

# In folgender Zeile bitte ihren Code eintragen

integer = 50

# Hiermit wird der Wert der Variable 'integer' ausgegeben

print(integer)

Klasse! Die Variable 'integer' ist korrekt!
Schade, die Variable 'integer' ist leider nicht korrekt!
