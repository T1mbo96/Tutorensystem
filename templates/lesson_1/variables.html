Variablen

Eine Variable im allgemeinsten Sinne ist ein einfacher Behälter zur Aufbewahrung von bestimmten Werten, also z.B. Buchstaben oder Zahlen. Man kann im Verlauf des Programms auf diese Variablen, oder genauer auf den Wert ihres Inhaltes zugreifen, oder ihnen einen neuen Wert zuweisen. Jede Variable benötigt außerdem einen eindeutigen Variablennamen (z.B. x oder y), sodass das Programm auch genau weiß, welche Variable es verwenden soll. Der zugehörige Block sieht folgendermaßen aus.

In diesem Beispiel kann man zwischen den Variablen x oder y wählen, welche im nächsten Spiel für die jeweilige x- bzw. y-Koordinate stehen.
