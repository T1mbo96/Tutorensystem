MauMau

Download-Link

Die erste Aufgabe der dritten Lektion besteht darin, zwei Funktionen in der vorgefertigten Klasse Card für das Spiel MauMau zu implementieren. Die erste Funktion überprüft, ob eine Karte, auf Grundlage einer vorher gespielten Karte, gespielt werden darf. Die zweite Funktion soll überprüfen, welche Karten von der Hand, auf Grundlage der vorher gespielten Karte, gespielt werden dürfen.

Gespielt wird mit einem Kartensatz, in welchem eine Karte jeweils durch eine Farbe (Hearts, Spades, Diamonds, Clubs) und einem Rang (7, 8, 9, 10, Jack, Queen, King, Ace) definiert ist.

Sie müssen sich für diese Aufgabe nicht mit dem gesamten Regelwerk von MauMau auskennen. Es reicht, wenn Sie wissen, dass eine Karte nur gespielt werden darf, falls Sie entweder dieselbe Farbe oder denselben Rang wie die zuvor gespielte Karte besitzt (oder beides).
def can_be_played_on(self, other: 'Card') -> bool:

Diese Funktion soll überprüfen, ob die aktuelle Karte auf eine übergebene Karte other gespielt werden darf.

Eine Karte darf genau dann gespielt werden, falls die Farbe, der Rang oder beides mit der anderen Karte übereinstimmt.

Falls die Karte gespielt werden darf, soll True zurückgegeben werden. Falls die Karte nicht gespielt werden darf, soll False zurückgegeben werden.
def playable_cards(card: Card, hand: List[Card]) -> List[Card]:

Diese Funktion überprüft, welche Karten aus einer gegebenen Liste hand auf die übergebene Karte card gespielt werden dürfen.

Gespielt werden dürfen Karten wieder analog zur ersten Funktion.

Zurückgegeben werden soll eine Liste, welche alle Karten enthält, die gespielt werden dürfen. Falls keine Karten gespielt werden dürfen, soll eine leere Liste zurückgegeben werden.
