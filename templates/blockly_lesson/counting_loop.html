Schleifen: Zählschleife

Bisher haben Sie die Schleifen nur so kennengelernt, dass die Spielfigur eine Aktion ausführt, bis sie einen Marker auf dem Spielfeld erreicht hat. Mithilfe dieser neuen Schleifenart können Sie Aktionen beliebig oft (z.B. 3-, 4- oder 5-Mal) ausführen lassen. Sie werden hierfür auch nicht mehr die sogenannte while-Schleife mit einer Schleifenbedingung verwenden, sondern eine for-Schleife. Wie genau diese funktioniert und was Sie noch alles mit dieser anstellen können, erfahren Sie in späteren Kapiteln. Aktuell ist für Sie nur wichtig zu wissen, dass Sie Aktionen nun mehrmals ausführen können, ohne an eine Schleifenbedingung gebunden zu sein.

Wie Sie hier erkennen, können Sie aus verschiedenen Möglichkeiten auswählen, wie oft der Schleifenblock do ausgeführt werden soll.
