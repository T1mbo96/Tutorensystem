Anweisungen

Eine Anweisung ist eine einzelne, in der Syntax der jeweiligen Programmiersprache formulierte, einzelne Vorschrift, welche bei der Abarbeitung des Programms ausgeführt wird.

Bedingte Anweisungen

Eine bedingte Anweisung stellt einen Programmabschnitt dar, der nur unter bestimmten Bedingungen ausgeführt wird.

Schleifen

Eine Schleife ist ebenfalls eine Kontrollanweisung, welche es ermöglicht, dass ein bestimmter Teil des Codes wiederholt ausgeführt wird.

Variablen

Eine Variable im allgemeinsten Sinne ist ein einfacher Behälter zur Aufbewahrung von bestimmten Werten, also z.B. Buchstaben oder Zahlen.

Logischer Operator: Und

Ein Operator ist ein Symbol zur Durchführung linguistischer, logischer oder mathematischer Operationen (z.B. '+' oder '-' oder 'und' oder 'oder').

Zählschleife

Mithilfe der Zählschleife können Sie Aktionen beliebig oft (z.B. 3-, 4- oder 5-Mal) ausführen lassen.

Funktionen

Zusammengesetzt ist eine Funktion aus dem Rückgabewert, dem Funktionsnamen und den Parametern, welche die Funktion erhält, um damit zu arbeiten. Allerdings muss eine Funktion nicht zwangsweise einen Rückgabewert oder Parameter besitzen.
