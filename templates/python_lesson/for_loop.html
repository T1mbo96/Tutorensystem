for-Schleife

Auch die for-Schleife konnten Sie in Lektion 1 schon ausprobieren und ihre Funktionsweise verstehen. Allerdings wurde diese Schleifenart in den Spielen nur als sogenannte Zählschleife verwendet, welche es per se so in Python nicht gibt.

Zählschleife

Bei dieser Variante einer Schleife muss die Schleifenvariable, also Startwert, Endwert und die Schrittweite numerisch sein. Das heißt, die Schleifenvariable wird auf einen Startwert gesetzt, welche anschließend, vom Startwert ausgehend, um die festgesetzte Schrittweite inkrementiert/dekrementiert/etc. wird. Dies geschieht so lange, bis die Schleifenvariable einen gewissen Endwert erreicht und die Schleife dadurch verlassen wird.

Wie oben erwähnt, gibt es diese Schleifenform in Python eigentlich nicht. Man kann dieses Verhalten aber durch die Benutzung der range-Funktion modellieren. Diese Funktion liefert einen Iterator, der eine gewisse Anzahl an Zahlen generieren kann.

Man kann auch einen Start- sowie Endpunkt definieren, sodass die Zahlen von begin (inklusive) bis end (exklusive) generiert werden.

Außerdem kann man die Schrittweite angeben, in welcher die Zahlen generiert werden sollen. Diese gibt man einfach nach begin und end an.

Somit kann man diese Funktion folgendermaßen nutzen, um die standardmäßige for-Schleife von Python als Zählschleife zu verwenden.

Die Variable counter ist hierbei die Schleifenvariable, welche den entsprechenden Wert in jedem Schleifendurchlauf annimmt. In diesem Beispiel also im ersten Durchlauf den Wert 0, im zweiten Durchlauf den Wert 2 und im dritten Durchlauf den Wert 4. Die Variable counter kann ebenfalls beliebig benannt werden.

Foreach-Schleife

Die standardmäßige for-Schleife in Python ist eine sogenannte foreach-Schleife. Diese Schleifenart dient der Iteration über eine Sequenz von Objekten (also dem Durchgehen aller Elemente), wie zum Beispiel den Zeichen in einem String. Sie kann ebenfalls einen else-Teil besitzen, äquivalent zu dem der while-Schleife. Dies bedeutet, dass man ebenfalls die Schlüsselwörter break und continue verwenden kann sowie, dass dieselben Regeln dafür gelten.
