Aufgabe: Strings

Erzeugen Sie bitte eine Variable mit Namen test_string, deren Wert Sie auf Hallo, Welt! setzen. Anschließend erzeugen Sie bitte drei weitere Variablen mit den Namen first_string, second_string und third_string. Weißen Sie first_string bitte den Wert des sechsten Elements, second_string den Wert des zweiten Elements und third_string den Wert des letzten Elements von test_string zu. Nutzen Sie zu Lernzwecken bitte die Zuweisung durch Indexierung der test_string Variable, wie Sie in vorhergehendem Abschnitt gelernt haben.

Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run ausführen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen (Löscht auch Ihren Code!). Außerdem wird eine Musterlösung für die Aufgabe durch einen Klick auf den Button Solution bereitgestellt.
Bitte Code hier eingeben:

# In den folgenden Zeilen bitte ihren Code eintragen

test_string = "Hallo, Welt!"
first_string = test_string[5]
second_string = test_string[1]
third_string = test_string[-1]

# Hiermit werden die Werte der Variablen ausgegeben

print(test_string)
print(first_string)
print(second_string)
print(third_string)
