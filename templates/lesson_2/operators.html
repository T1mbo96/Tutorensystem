Operatoren

Ein Operator ist ein Mittel, Verfahren o.Ä. zur Durchführung linguistischer, logischer oder mathematischer Operationen. Er verknüpft zwei oder mehrere Operanden (Werte, z.B. Variablen oder Konstanten) zu einem neuen Wert.

Arithmetische Operatoren

Bezeichnung Zeichen Beispiel
Addition + x + y
Subtraktion - x - y
Multiplikation * x * y
Division / x / y
Modulo (Rest beim ganzzahligen Teilen) % x % y
Ganzzahldivision (Ganzzahliger Anteil der Division) // x // y
Potenzieren ** x ** y
Negatives Vorzeichen - -x

Vergleichsoperatoren

Bezeichnung Zeichen Beispiel
größer als > x > y
kleiner als < x < y
größer oder gleich >= x >= y
kleiner oder gleich <= x <= y
gleich == x == y
ungleich != x != y

Merke: Wir weisen einer Variable einen Wert mit einem = zu, prüfen zwei Variablen aber mit zwei == auf Gleichheit.

Logische Operatoren

Bezeichnung Zeichen Beispiel
Boolesches Oder or x or y
Boolesches Und and x and y
Boolesches Nicht not x not y

Membership Operatoren

Bezeichnung Zeichen Beispiel
"Element von..." in x in y
"Nicht Element von..." and x not in y

Identity Operatoren

Bezeichnung Zeichen Beispiel
"...zeigt auf dasselbe Objekt wie..." is x is y
"...zeigt nicht auf dasselbe Objekt wie..." is not x is not y

Merke: Die Nutzung der Membership und Identity Operatoren wird in späteren Abschnitten ersichtlich, wenn Sie Container wie z.B. Listen und objektorientiertes Programmieren kennen lernen. Außerdem gibt es noch sogenannte Bitweise Operatoren. Diese werden in unseren Aufgaben allerdings nicht benötigt. Falls sie diese gerne in Erfahrung bringen würden, dann können Sie dies mithilfe des dritten Links Python Operatoren 3.
