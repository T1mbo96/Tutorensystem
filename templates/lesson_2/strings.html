Strings

Ein String gehört zu den sogenannten sequentiellen Datentypen. Diese enthalten eine Folge von gleichartigen oder verschiedenartigen Elementen, welche eine definierte Reihenfolge aufweisen. Aus diesem Grund kann man mithilfe von Indices auf die Elemente, die in einem sequentiellen Datentyp gespeichert sind, zugreifen. Ein String im Speziellen ist eine Zeichenkette, die durch das in Anführungszeichen setzen von Zeichen erzeugt werden.

Wie man in obiger Abbildung gut erkennen kann, werden die Elemente mit 0 beginnend, von links nach rechts, indexiert. Man kann auf einen sequentiellen Datentyp allerdings auch von rechts beginnend nach links zugreifen, woraus sich dann eine Indexierung von -1 absteigend ergibt. Somit kann man jedes Element direkt einzeln ansprechen.

Es gibt einige Möglichkeiten, was man mit Strings im Speziellen bzw. sequentiellen Datentypen im Allgemeinen alles anstellen kann. Um eine Zusammenfassung über die Möglichkeiten zu erhalten, können Sie sich den ersten Link Python Strings 1 ansehen.
