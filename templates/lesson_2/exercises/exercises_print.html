Aufgabe: Print-Funktion

Geben Sie bitte zuerst die Variablen one und two mithilfe der Print-Funktion standardmäßig in einem Funktionsaufruf aus. Anschließend sollen die Variablen one und three mit einem & (und) getrennt ausgegeben werden. Zum Schluss soll dann noch die Variable one mit einem ! am Ende ausgegeben werden, ohne Zeilenumbruch.

Die Reihenfolge der Ausgabe ist wichtig! Also arbeiten Sie die Aufgabe bitte von oben nach unten durch und geben Sie bitte das Geforderte auch in dieser Reihenfolge aus!
Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run testen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen.
Bitte Code hier eingeben:

Klasse! Zeile 1 ist korrekt!
Klasse! Zeile 2 ist korrekt!
Klasse! Zeile 3 ist korrekt
Schade, Zeile 1 ist leider nicht korrekt!
Schade, Zeile 2 ist leider nicht korrekt!
Schade, Zeile 3 ist leider nicht korrekt!
