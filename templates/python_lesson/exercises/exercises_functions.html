Aufgabe: Funktionen

Nun sollen Sie die erste größere Aufgabe dieser Lektion bearbeiten. Dafür implementieren Sie bitte das Sortierverfahren Bubblesort, welches eine gegebene Liste in die richtige Reihenfolge bringen soll (Zahlenwerte aufsteigend sortiert).

Bei Bubblesort wird jeweils in Runden für jedes benachbarte Paar von Elementen getestet, ob diese vertauscht werden müssen. Sobald mindestens eine Vertauschung notwendig war, beginnt eine neue Runde. War keine Vertauschung notwendig, dann ist die Liste korrekt sortiert.


Von Swfung8-Eigenes Werk,CC BY-SA 3.0,Link

Seinen Namen hat das Sortierverfahren der Bewegung der größeren Elemente nach und nach von links nach rechts zu verdanken. Sie steigen somit ähnlich wie Blasen im Wasser nach oben.

Erstellen Sie nun also eine Funktion def bubblesort(unordered_list): und implementieren Sie das Sortierverfahren in dieser. Die Funktion soll die sortierte Liste abschließend zurückgeben.

# In den folgenden Zeile bitte ihren Code eintragen

def bubblesort(unordered_list):
  unordered = True

  while unordered:
   unordered = False

   for counter in range(len(unordered_list) - 1):
    if counter < len(unordered_list) - 1:
      if unordered_list[counter] > unordered_list[counter + 1]:
       temp = unordered_list[counter]
       unordered_list[counter] = unordered_list[counter + 1]
       unordered_list[counter + 1] = temp
       unordered = True

  return unordered_list

# Hiermit werden die Listen ausgegeben

print(bubblesort(unordered_list1))
print(bubblesort(unordered_list2))
print(bubblesort(unordered_list3))
print(bubblesort(unordered_list4))
