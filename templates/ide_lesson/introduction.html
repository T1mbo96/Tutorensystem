Einführung

Zum Abschluss dieses Tutorials dürfen Sie nun Aufgaben in einer eigens gewählten IDE lösen. Dafür werden Ihnen zwei Guides zur Verfügung gestellt. Einmal für die IDE PyCharm und einmal für die IDE IPython. Der Unterschied besteht darin, dass PyCharm eine lokale IDE darstellt und IPython im Browser verwendet wird. Somit können neue Module in IPython einfach importiert werden, wohingegen neue Module in PyCharm erst installiert werden müssen.

Ihnen ist natürlich freigestellt jede andere IDE zu wählen, die Sie möchten, allerdings werden sich Beispiele und Erklärungen nur auf diese zwei IDE's beziehen, weshalb Ihnen empfohlen wird, sich für eine von diesen beiden zu entscheiden.
