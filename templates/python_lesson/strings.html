Strings

Ein String ist eine Sequenz von einzelnen Zeichen, weshalb dieser Typ auch als Zeichenkette bezeichnet wird. Man kann auf die einzelnen Zeichen mithilfe von Indizes zugreifen.

Wie man in obiger Abbildung gut erkennen kann, werden die Elemente mit 0 beginnend, von links nach rechts, indexiert. Man kann allerdings auch von rechts beginnend nach links zugreifen, woraus sich dann eine Indexierung von -1 absteigend ergibt. Somit kann man jedes Element direkt einzeln ansprechen.

Es gibt einige Möglichkeiten, um mit Strings zu arbeiten. Um eine Zusammenfassung über diese Möglichkeiten zu erhalten, können Sie den ersten Link Python Strings 1 verwenden.
