Datentypen

Ein Datentyp bezeichnet die Zusammenfassung konkreter Wertebereiche und die darauf definierten Operationen zu einer Einheit. Somit können Variablen von einem bestimmten Typ sein, woraus sich dann die Möglichkeiten ergeben, wie diese verwendet werden können bzw. welche Operationen mithilfe oder auf dieser Variable ausgeführt werden dürfen. In Python wird die sogenannte dynamische Typisierung verwendet, somit erfolgt die Typzuteilung der Variablen zur Laufzeit eines Programms durch das Laufzeitsystem. Dies bedeutet, dass der Wert einer Variable auch gleichzeitig ihren Typ bestimmt. Der Typ einer Variable kann außerdem mithilfe einer speziellen Funktion in einen anderen umgewandelt werden.

Numerische Datentypen

Bezeichner Bedeutung Konvertierung
Integer Ganzzahl Funktion int()
Long Integer Ganzzahl mit beliebiger Länge Funktion long()
Float Gleitkommazahl Funktion float()
Komplexe Zahlen als zwei Fließkommazahlen dargestellt Funktion complex()
Boolescher Typ Wahrheitswert Funktion bool()

Der boolesche Datentyp wird zu den numerischen Datentypen gezählt, da alle arithmetischen Operationen, die im Abschnitt Operatoren definiert sind, angewendet werden können. Außerdem können zusätzlich auch noch die logischen Operatoren not, and und or auf booleschen Variablen angewendet werden.
