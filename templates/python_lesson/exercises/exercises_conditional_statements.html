Aufgabe: Bedingte Anweisungen

Schreiben Sie eine bedingte Anweisung, in der für jedes Punkteintervall der nachfolgenden Tabelle der entsprechende Preis ausgegeben wird, sodass ein Wettbewerber wissen kann, welchen der Preise er aus der Tabelle gewonnen hat. Die Auswahl des Preises basiert auf der Anzahl der erzielten Punkte, die in der ganzzahligen Variable points gespeichert ist.

Points Preis
1-80 Kuscheltier
81-130 Ferngesteuertes Auto
131-170 Kein Preis
171-200 Puzzle

Alle unteren und oberen Grenzen sind hier inklusive und Punkte können nur positive ganzzahlige Werte bis zu 200 annehmen.
Erzeugen und weisen Sie in Ihrer bedingten Anweisung der Variable result eine Zeichenfolge (String) zu, die, basierend auf dem Wert der Punkte, die entsprechende Nachricht enthält. Wenn ein Preis gewonnen wurde, sollte die Nachricht Sehr gut! Sie haben ein [Preisname] gewonnen! ausgegeben werden (Ersetzen Sie [Preisname] mit dem Namen des jeweiligen Preises aus der Tabelle). Wenn es keinen Preis gibt, sollte in der Nachricht Oh schade, diesmal gibt es keinen Preis. stehen.

Wenn Sie ihren Code eingegeben haben können Sie ihn mit einem Klick auf den Button Run testen. Um eine Rückmeldung zu erhalten, ob ihr Code richtig ist, benutzen Sie bitte den Button Test. Falls Sie noch einmal von vorn beginnen wollen, oder ausversehen Code der Aufgabe gelöscht haben, dann können Sie die Aufgaben einfach mit einem Klick auf den Button Reset zurücksetzen.

Sie dürfen ihren eigenen Code gerne mit anderen Eingaben testen. Um den Test bestehen zu können, sollten Sie zum testen nur den ursprünglichen Wert von 174 nutzen.
Bitte Code hier eingeben:

Klasse! Sie haben den richtigen Preis korrekt ausgegeben!
Schade, der Preis oder die Nachricht ist leider nicht korrekt!
