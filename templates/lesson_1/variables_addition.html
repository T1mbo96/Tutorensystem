Variablen: Zusatz

Am Ende des Schildkröten-Spiels in Level 10 dürfen Sie eine Vielzahl an neuen Blöcken selbst ausprobieren, ohne irgendwelche Ziele erreichen zu müssen. Dazu wird hier und im nächsten Abschnitt nur eine kleine Hilfestellung zu den Variablen- und Funktionsblöcken gegeben.

Create

Um eine Variable zu erstellen, klicken Sie einfach in der Toolbox auf den entsprechenden Reiter Variables und wählen Sie Create variable... aus.

Im anschließend aufgehenden Textfenster müssen Sie ihrer Variable nurnoch einen Namen geben, sodass Sie sie danach benutzen können.

Assign

Um einer Variable einen Wert zuzuweisen, wählen Sie den Block set [Variablenname] to aus und hängen einen entsprechenden Block an (z.B. einen einfach Block mit einem Zahlenwert aus dem Math-Reiter).

Update

Wenn Sie den Wert einer Variable ändern möchten, dann benutzen Sie den Block change [Variablenname] by [Wert].

Standardmäßig wird ein Zahlenwert erwartet, allerdings können Sie auch andere Blöcke an den Block hinten anhängen.

Use

Zur Benutzung der erstellten Variablen, verwenden Sie einfach den Block den Sie im Abschnitt Variablen bereits kennengelernt haben.
