Aufgaben

Nachdem Sie alles Notwendige installiert haben, könnten Sie jetzt eigentlich beginnen die ersten Aufgaben zu lösen. Allerdings fehlen noch ein paar wichtige Informationen zu Modulen, Namensräumen und Ausnahmebehandlung. Diese drei Themen werden Ihnen in den folgenden Abschnitten beigebracht. Außerdem folgt hier noch eine Einführung, wie sie den Code und die Files für die Tests der Aufgaben herunterladen und in PyCharm aufrufen können.
Der zugehörige Code und die zugehörigen Tests für jede Aufgabe befinden sich jeweils in einem Repository auf GitHub. Hierfür finden Sie im Abschnitt der Aufgabe einen Link zum Repository, der ähnlich zu folgendem aufgebaut ist.

https://github.com/PythonTut/example.git

Diesen Link kopieren Sie und folgen den weiteren Schritten. Falls Sie das Repository von der Startseite aus klonen wollen, dann wählen Sie Folgendes aus.

Falls Sie sich bereits in einem Projekt befinden und ein neues Projekt klonen wollen, dann wählen Sie das Folgende aus.

Nachdem Sie einem von beiden Schritten gefolgt sind, erscheint folgendes Fenster, in welchem Sie den Link der Aufgabe in das Feld URL einfügen.

Abschließend müssen Sie nurnoch Clone auswählen und das geklonte Projekt öffnen.
